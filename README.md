# nixos-configs

My Config Files For Various NixOS Deployments

## Pull bootstrap.sh & Run Script

```bash
curl -L https://github.com/Wolfereign/nixos-configs/tarball/master | tar xz
cd Wolfe*
bash ./bootstrap.sh
```

or 

```bash
curl -L https://git.io/fNeXc | tar xz
cd Wolfe*
bash ./bootstrap.sh
```
